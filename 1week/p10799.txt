1. 문제 인식

쇠막대기와 레이저의 배치를 나타내는 입력이 괄호 문자열로 주어집니다. 레이저는 여는 괄호와 닫는 괄호의 인접한 쌍인 ()로 이루어지며, 쇠막대기는 왼쪽 끝이 (로 표현되며 오른쪽 끝이 )로 표현됩니다.
이때, 잘려진 쇠막대기 조각의 개수를 구하는 문제입니다.

2. 문제 유형 파악

쇠막대기를 레이저로 자르면 현재 있는 쇠막대기 수만큼 쇠막대기 개수가 증가합니다.
그러므로 여는 괄호 ( 를 만났을 때 push하여 쇠막대기 개수를 표현하고 레이저를 만났을때 현재 스택의 사이즈 만큼 결과값에 더해주면 되므로, 스택 자료구조를 사용합니다.

3. 다이어그램 또는 알고리즘을 나열

    1) 괄호 문자열을 입력받음.
    2) 괄호 문자열의 길이만큼 반복문을 돌며 i+1함. 반복문 안에선 다음과 같은 조건을 검사함.
        (1) 스택이 비어있지 않다면 아래 조건문을 수행함.
            1. 괄호 문자열의 i번째 문자가 ( 일때, 스택에 push하고 결과값을 저장하는 변수인 res에 +1함.
            2. 괄호 문자열의 i번째 문자가 ) 이고, i-1번째 문자가 ( 라면 res에 -1하고, pop을 한번 수행 후, 현재 스택의 사이즈만큼 res에 더해줌.
            3. 괄호 문자열의 i번째 문자가 ) 이고, i-1번째 문자가 ) 라면 pop을 수행함.
        (2) 스택이 비어있다면 아래 조건문을 수행함.
            1. 스택에 push하고, res에 +1함.
    3) 반복문이 끝난후, res를 출력함.
